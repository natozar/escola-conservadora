## Voice & Tone

### General Rules
```
DO:
  - Usar "voce" (informal, acolhedor)
  - Frases curtas e diretas
  - Celebrar conquistas do aluno
  - Usar linguagem positiva e encorajadora
  - PT-BR padrao (sem regionalismos extremos)
  - Ser educativo sem ser pedante

DON'T:
  - Usar "tu", "vos", "senhor"
  - Usar jargao educacional pesado
  - Ser condescendente ou infantilizar
  - Usar pressao emocional excessiva
  - Fazer promessas irrealistas
  - Usar superlativos vazios ("o melhor", "numero 1")
```

### Tone per Context
```
Onboarding:    Animado, acolhedor ("Bem-vindo! Vamos comecar?")
Lessons:       Didatico, claro ("Vamos entender como...")
Quizzes:       Desafiador, divertido ("Hora do desafio!")
Achievements:  Celebratorio ("Parabens! Voce desbloqueou...")
Errors:        Empathico, util ("Ops! Tente de novo")
Marketing:     Confiante, aspiracional ("Seu filho merece...")
Legal:         Formal, claro (padrao juridico brasileiro)
```

### Language Rules
```
Bilingual strategy:
  - UI principal: Portugues brasileiro
  - Traducao completa: Ingles (i18n.js)
  - Codigo: Ingles (variaveis, funcoes, commits)
  - Documentacao tecnica: Ingles
  - Documentacao usuario: Portugues
```
